# Common Eclipse Refactoring Framework — Update Site

This branch hosts the p2 update site via GitHub Pages.

**Update site URL (composite, all versions):**
`https://ponder-lab.github.io/Common-Eclipse-Refactoring-Framework/releases/`

`releases/` is a p2 composite repository. `releases/archive/` holds all
versions released through 5.3.0 (seeded from the legacy in-repo update site);
future releases are published as their own `releases/X.Y.Z/` children.

Published by `.github/workflows/release.yml`. Do not edit by hand.
